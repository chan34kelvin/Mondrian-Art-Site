# Mondrian Art Site

A mondrian style layout site that shows its art style and allows people to adjust the pattern inside.

## What's in this project?

‚Üê `README.md`: Details on what's on this site and what's it about.

‚Üê `index.html`: The main page that shows mondrian art with the original colors.

‚Üê `style.css`: holds the css for navbar and every non-grid related elements on this site.

‚Üê `m-color.css`: mondrian color css files that contains var which describe the behaviors of background-image during index.html when asked in m.css.

‚Üê `m-image.css`: mondrian image css files that contains var which link images to the background-image property during image-page.html when asked in m.css
